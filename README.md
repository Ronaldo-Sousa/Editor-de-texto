# Editor-de-texto
Criando um editor de texto simples em C# para abrir, escrever e salvar documentos. O algoritmo começa com um menu simples para tornar a experiência mais intuitiva ao usuário, mostrando quais são as opções disponíveis e com uma estrutura de seleção "switch case" para direcionar o usuário de acordo com o que ele escolher. Caso a opção seja de abrir um arquivo existente, o programa imprime na tela uma mensagem perguntando onde está o arquivo, então o usuário digita a localização do arquivo nas pastas, o algoritmo encontra o arquivo e abre ele. Caso o usuário opte por criar um arquivo novo, o algoritmo abre uma interface para o usuário digitar o que quer e ao final ele captura o texto digitado para ser salvo, perguntando ao usuário onde salvar o arquivo, nesse momento o usuário digita o local onde quer que o arquivo seja salvo e o algoritmo encontra esse local e salva o arquivo. O algoritmo usa os métodos "StreamWriter" para permitir digitar um texto e "StreamReader" para ler um texto existente, eu gosto de chamá-los de "caneta" e "óculos" para fins didáticos, essas duas estruturas são fundamentais para o funcionamento do código, bem como a estrutura "using()", que possibilita abrir e editar um arquivo e o fecha automaticamente depois que você termina a sua ação.
